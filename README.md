# Unexamined

* Artwork and code by Jeni Dhodary

# About

This repository contains the source code for *Unexamined*, a personal project tracing the unrecognized, irrational threshold of attachment between a human and a machine.

The implementation and the idea are both integral to the work.

You are welcome to study and learn from this project in non-commercial contexts, provided appropriate credit is given.
